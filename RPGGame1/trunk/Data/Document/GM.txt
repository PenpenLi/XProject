1.添加物品:
additem type id num bind
type: 物品类型:
	eProp = 1,		--道具
	ePet  = 3, 		--宠物
	ePartner = 4,   --伙伴
	eFaBao = 5,		--法宝
		
id: 物品ID
num: 物品数量
bind: 是否绑定(0否, 1是)

例子：
增加银两:additem 1 31004 10

2.重新加载所有服务脚本:
reloadall

3.添加宠物
addPetObj 101
101为宠物编号 

4.添加宠物寿命
addLife nPos value
nPos为宠物位置，value添加的值

5.添加宠物经验
addExp nPos value

6.添加好友友好度
wgm addfriendhoney 好友角色ID 友好度数量

7.发送邮件
sendmail 物品类型 物品ID 物品数量

19.开启活动
openact id min subid round propid
id 活动ID(HuoDongConf.xml)
min 开启分钟数
subid 子活动ID(限时奖励有)
ext1 扩展参数1(轮次/其他)
ext2 扩展参数1(替换物品ID)

19-1.关闭活动
closeact id subid
id 活动ID(HuoDongConf.xml)
subid 子活动ID(限时奖励有)

19-2.关闭所有活动
closeallact

35.开启限时活动(决战九霄,混沌试炼),单位时间内不能多开
openpveact nAcdID, nReadyMin, nEndMin
nAcdID--活动ID
nReadyMin--大厅准备时长(默认为5分钟)
nEndMin--活动持续时长(默认为一个小时)

36.关闭限时活动
